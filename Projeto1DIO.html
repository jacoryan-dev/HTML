O que é HTML?

Voltar para o topo

HTML, sigla para HyperText Markup Language ou, em português, Linguagem de Marcação de Hipertexto, é a linguagem fundamental para a criação de páginas da web. Imagine-a como o esqueleto de uma casa: ela define a estrutura básica, onde cada elemento se encaixa e como eles se relacionam.

Em termos mais técnicos, o HTML utiliza tags para marcar o conteúdo de um documento, indicando ao navegador como ele deve ser interpretado e exibido na tela. Essas tags são palavras-chave cercadas por sinais de menor e maior (< e >) chamadas de tag, como por exemplo: <h1>, <p>,<img>.

Estrutura básica de um HTML

<!DOCTYPE html>
<html>

<head> <title>Título da Página</title> </head>

<body>

<h1>Meu cabeçalho</h1>

<p>Meu Parágrafo.</p>

</body>

</html>

Diferença entre listas ordenadas e não ordenadas

No HTML utilizamos dois tipos de listas: ordenadas e não ordenadas. A diferença é que na primeira a ordem importa já na segunda a ordem é desconsiderada.

Listas Ordendas (<ol>)


Os itens de uma lista ordenada são numerados sequencialmente, indicando uma ordem específica ou hierarquia.
Ideal para instruções, etapas de um processo, rankings, cronologias, etc.



Listas Não Ordenadas (<ul>)


Os itens de uma lista não ordenada são marcados com um símbolo (geralmente um ponto), indicando que a ordem não é crucial.
Ideal para listas de itens que não possuem uma ordem intrínseca, como uma lista de ingredientes, itens de um menu, etc.
